O que é?

O dreamCode é um framework escrito em PHP para ajudar no desenvolvimento de sistemas e sites para internet, tornando o desenvolvimento mais ágil e agradável.

 

Particularidades

O dreamCode leva em seu código uma série de estudos e particularidades que foram notadas ao longo de anos de desenvolvimento de vários projetos.

Hoje, existem muitos padrões de projetos para se criar sistemas para internet, mas muitos ou quase nenhum deles leva em conta as particularidades da linguagem. O dreamCode foi escrito levando em conta os seguintes itens:

1) Implementação dos comandos mais rápidos do PHP.

Estudos comparativos de comandos echo, print, for, while e etc. foram feitos para tornar o framework o mais rápido possível na execução de seu código.

2) Não foi inventado o que já existe.

Muitos framework inventam comandos para instanciar classes ou funções, neste framework tudo foi escrito o mais próximo possível do que já existe na linguagem PHP.

3) Reaproveitamento de código.

O dreamCode usa OOP para separar o layout do código (o modelo de negócio), tornando reaproveitavel e facil manutenção de uma aplicação.
